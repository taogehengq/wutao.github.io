Agent 全面技能（Level-2/3）

1. 语言与生态

  • Python 异步/协程
  • FastAPI / Flask
  • Node.js + LangChain.js

2. 算法与微调

  • LoRA / PEFT 微调
  • RLHF（强化学习对齐）
  • 多模态（CLIP、Whisper）

3. 记忆系统

  • 向量数据库（Milvus, Pinecone）
  • 图数据库（Neo4j）
  • 缓存（Redis 语义缓存）

4. Planning 策略

  • ReAct / Reflexion / Plan-and-Execute
  • 多 Agent 协作（CrewAI、AutoGen）
  • 状态机与工作流引擎

5. 工具与集成

  • Function Calling（OpenAI JSON Mode）
  • MCP 协议（Model-Context-Protocol）
  • API Gateway、OAuth、Webhook

6. 安全与合规

  • Prompt 注入防御
  • PII 脱敏、审计日志
  • RBAC 权限模型

7. 运维与评测

  • LLMOps（CI/CD、A/B、灰度）
  • 指标：任务成功率、Token 成本、延迟
  • 监控：LangSmith、Datadog、Prometheus
